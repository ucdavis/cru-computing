Application Development Assistant

This position will work with the College of Agricultural and Environmental Sciences Dean’s Office Computing Resources Unit (CRU) actively developing web applications and programs to be used in production situations campus-wide. Student Application Developers gain real-world experience designing, developing and testing applications utilizing modern computer languages and frameworks.


Student application developers are expected to have experience in at least one web programming language (e.g. C#, PHP, Python, Ruby, Java). Basic knowledge of web infrastructure including HTTP, HTML, server/browser interactions and familiarity with source code revision systems such as Git, Mercurial or Subversion is preferred. Experience with JavaScript highly desirable, familiarity with browser-based debugging tools for JS, CSS, network activity desirable, experience with ASP.NET, C#, and MVC frameworks desirable. Understanding of APIs, modern CSS practices and frameworks (Sass, Bootstrap) desirable. Punctuality, accountability, and attention to detail required.


To apply, visit https://ucdavis-csm.symplicity.com/ and search for 816951 (the positionID).


Website Development Assistant

The student assistant will assist the web team at the CA&ES Dean’s Office with daily responsibilities of web upkeep and long term projects. The student assistant will have the opportunity to contribute to design and development decisions, acquiring real world experience with both the creative and managerial aspects of web development. The ideal student assistant will be self-directed and motivated, as well as creative at problem solving.


We are seeking one detail-oriented and well-organized student assistant with an interest in web development. Applicants are expected to have experience in HTML and CSS. Experience with the Adobe Creative Suite and Sketch is desirable. Understanding of modern frameworks and css practices (Sass, Bootstrap) is desirable. Familiarity with browser-based debugging tools for HTML and CSS is desirable. Experience with Drupal is highly desirable. Applicants who are familiar with hosting, DNS services and have created multiple websites are highly desirable. Solid writing, communications and organizational skills required.


To apply, visit https://ucdavis-csm.symplicity.com/ and search for 817180 (the positionID).
